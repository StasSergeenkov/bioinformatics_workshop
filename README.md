# bioinformatics_workshop
## Задумка

Работа будет состоять из нескольких основных этапов:
1. Подготовка исходных данных: проверка качества сборки, аннотация
2. Общий анализ филогении: выделение 16s и построение филогенетического дерева
3. Поиск и сравнительный поиск с гомологами генов целлюлаз
4. Предсказание свойств найденных белков
5. Молекулярный докинг для найденных белков с раззличными типами субстратов

## Принцип работы с директориями
Все исходные, промежуточные и итоговые данные хранятся каталогах этой директории. Для каждого основного этапа создан свой каталог:
+ Подготовка исходных данных
+ Общий анализ филогении
+ Поиск и сравнительный поиск с гомологами генов целлюлаз
+ Предсказание свойств найденных белков
+ Молекулярный докинг

Внутри каждого каталога присутствует свой файл readme.md с подробным описанием структуры каталогов и операций проведённых с данными.
